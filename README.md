# Arale-Boilerplate

-----

基于阿里Arale前端模块架构，根据其开发规范（组件开发规范、文件目录规范等），形成的基础代码部署结构

-----

## 使用说明

该代码部署模板可直接 clone 至本地，然后根据项目实际需求调整，用于组件化开发
